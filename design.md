# お弁当予約アプリ - デザイン設計書

## アプリコンセプト
学生がお弁当を手軽に予約できるシンプルで使いやすいモバイルアプリ。
日時・メニューを選択して予約し、管理者はスプレッドシートで一覧を確認できる。

## カラーパレット
- Primary: #2E7D32 (深緑 - 食べ物・自然を連想)
- Background: #FFFFFF / #121212 (ダーク対応)
- Surface: #F1F8E9 / #1E2A1E
- Accent: #FF8F00 (オレンジ - 食欲を刺激)
- Success: #43A047
- Error: #E53935

## 画面一覧

### 1. ホーム画面 (index.tsx)
- 今日の日付・挨拶メッセージ
- 「予約する」ボタン (大きく目立つ)
- 直近の予約一覧 (カード形式)
- 予約件数バッジ

### 2. 予約フォーム画面 (reserve.tsx)
- 学生名入力フィールド
- 学年・クラス選択 (ピッカー)
- 受け取り日付選択 (DatePicker)
- 受け取り時間選択 (昼休み/放課後)
- メニュー選択 (カード形式グリッド)
  - 日替わり弁当
  - のり弁当
- 数量選択
- 備考入力
- 「予約する」ボタン

### 3. 予約一覧画面 (reservations.tsx)
- 予約リスト (FlatList)
- 日付フィルター
- キャンセルボタン
- スプレッドシートエクスポートボタン

### 4. エクスポート確認モーダル
- エクスポートプレビュー
- CSV/Excel形式選択
- 共有ボタン

## キーユーザーフロー
1. ホーム → 「予約する」タップ → 予約フォーム
2. フォーム入力 (名前・日時・メニュー) → 「予約する」タップ → 完了アニメーション → ホームへ戻る
3. ホーム → 「予約一覧」タブ → 一覧表示 → 「エクスポート」タップ → CSV共有

## タブ構成
- タブ1: ホーム (house.fill)
- タブ2: 予約する (calendar.badge.plus)
- タブ3: 予約一覧 (list.bullet.clipboard.fill)
